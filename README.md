## О Программе
**VKSearchSystem**

Программа выполняет автоматическое дополнение запросов для поисквой системы. На введённый пользователем запрос возвращается не более 5 наилучших подсказок.

К программе прилагаются две таблицы MySQL: **requestbase_requests** и **requestbase_pointer**.

Система инициализируется списком фраз на естественном языке, которые хранятся в таблице **requestbase_requests**. Вместе со списком фраз в этой таблице хранится количество обращений к этим запросам. 

Ранжирование подсказок осуществляется путем поиска запросов, начало которых совпадает с запросом, введенным пользователем, по максимальному количеству обращений к ним (столбец **request_amount**).

Введенный пользователем запрос сохраняется в таблице **requestbase_requests**. В случае, если введенный пользователем запрос имеется в этой таблице, счетчик количества обращений в этому запросу увеличивается на единицу.

Для оптимизации работы поиска в программе используется указатель. Он представляет собой таблицу **requestbase_pointer**, которая содержит в себе уникальные первые буквы и цифры строк, а также их номера в таблице **requestbase_requests**. 

Указатель автоматически дополняется и обновляется. При первом запуске программы, в случае если таблица **requestbase_pointer** не заполнена, неоходимо снять комменаций с запуска метода "pointerBaseInit" на 46-й строке класса SearchSystemApp.

Для работы программы достаточно запустить основной класс и ввести в консоль любую фразу. Программа работает при наличии не более одной опечатки, начиная со второго символа в запросе.

С уважением!
